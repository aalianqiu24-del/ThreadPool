项目描述：本项目实现了一个高性能、可复用的C++线程池组件，采用C++11标准库原生多线程支持，通过任务队列与工作线程池的
协作机制，有效解决了多任务并发执行时的线程创建销毁开销问题，大幅提升了系统的并发处理能力和资源利用率。
• 在线程管理层面，采用固定数量的工作线程模式，利用
std::thread创建线程组，通过
std::atomic<bool>原子标志控制线程的生命周
期，避免了频繁创建销毁线程带来的性能损耗，实现了线程资源的高效复用。
• 在任务调度方面，设计并实现了线程安全的无锁同步队列
SyncQueue，基于
std::mutex互斥锁和
std::condition_variable条件变量实
现生产-消费者模型，支持任务的异步提交和执行，确保了多线程环境下的数据一致性和线程安全。
• 充分利用现代C++编程特性，通过函数封装、值语义优化、回调简化、资源自动化管理以及线程安全的单次初始化机制，构建出简
洁高效且易于扩展的并发编程框架，充分体现了现代C++在系统级开发中的表达力与安全性。
